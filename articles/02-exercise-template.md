## Exercise title
Short description.

### Goal
- Clear, testable outcome (what the function should return)

### Starter
- `examples/<slug>/` contains starter code and tests

### How to submit
1. Fork the repo
2. Create a branch `exercise/<your-name>`
3. Open a pull request with passing tests

### Grading rubric
- Correctness: 60%
- Tests & edge-cases: 20%
- Code clarity & README: 20%
